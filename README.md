# PCA Cancer Dataset Analysis

## Overview
This project applies Principal Component Analysis (PCA) to the breast cancer dataset from sklearn to identify essential variables and reduce dimensionality.

## Objectives
- Perform PCA on the dataset
- Reduce data to 2 principal components
- (Optional) Apply logistic regression for prediction

## Technologies Used
- Python
- pandas
- numpy
- matplotlib
- scikit-learn

## Steps
1. Load dataset using sklearn
2. Standardize the data
3. Apply PCA (2 components)
4. Visualize results
5. Train logistic regression model (optional)

## How to Run
bash
pip install numpy pandas matplotlib scikit-learn
python pca_analysis.py
